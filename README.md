# ポイント管理Webアプリ

HTML / CSS / JavaScript を用いて制作したポイント管理Webアプリです。
実生活での子どものお小遣い管理をきっかけに開発を開始し、現在は利用者ごとにさまざまな使い方ができる汎用的なポイント管理アプリとして継続開発を行っています。

---

## ■概要

本プロジェクトは、子どものお手伝いや学習に対してポイントを付与し、一定ポイント到達時にお小遣いを支給する仕組みをデジタル化する目的で制作しました。

当初は家庭内利用を想定していましたが、開発を進める中で利用者ごとに異なる使い方ができる柔軟なアプリを目指し、機能を拡張しています。

---

## ■制作背景

子どもへのお小遣い支給を開始する際、定額制ではなく「行動に対する対価」としてお金のありがたみを学んでほしいと考えました。

そこで、

- お手伝い
- 学習
- 家庭内の役割

などに対してポイントを付与し、一定ポイント到達時にお小遣いを支給する仕組みを採用しました。

しかし、

- ポイントカードの紛失
- 現在のポイント数が分かりづらい
- 家庭内で運用ルールが変化した

という課題があり、それらを解決するため本アプリを制作しました。

また、家庭内だけでも支給ルールが何度か変化した経験から、利用者ごとに自由な使い方ができるアプリの方が価値があると考え、汎用的なポイント管理アプリへと発展させています。

## ■主な機能

### ● ユーザー管理

- ユーザー追加・編集・削除
- ユーザー切り替え機能

### ● 項目管理

- 項目追加・編集・削除

### ● ポイント管理

- ポイント加算
- ポイント減算
- ポイント支給
- ポイントリセット
- 現在ポイント表示
- 累計ポイント表示

### ● 履歴機能

- 直近5件の履歴表示
- 全履歴表示

### ● 目標機能

- 個人目標設定
- 共有目標設定
- 進捗バーによる達成状況の可視化
- 目標機能のON/OFF設定

### ● 設定機能

- マイナス残高許可ON/OFF
- 削除確認ON/OFF
- エラーメッセージ表示
- アプリデータ初期化

### ● データ保存

- LocalStorageによるデータ永続化

## ■工夫した点

### 1. 幅広い利用シーンに対応できる設計

当初は子どものお小遣い管理を目的としていましたが、
利用者ごとに異なる使い方ができるよう、
ユーザーや項目を自由に追加・編集・削除できる構成にしました。

### 2. モチベーションを維持する目標機能

個人目標と共有目標を実装し、
ポイントを貯める楽しさや達成感を得られる仕組みを意識しました。

### 3. 進捗の可視化

進捗バーを導入し、
特に子どもでも現在の達成度を直感的に理解できるよう工夫しました。

### 4. 柔軟なカスタマイズ性

機能のON/OFF設定を実装し、
利用者ごとの運用方法に合わせてアプリの使い方を変えられる設計にしています。

---

## ■苦労した点

当初は子どものお小遣い管理アプリとして開発を進めていましたが、途中で汎用的なポイント管理アプリへ方向転換したため、データ構造の見直しが必要になりました。

機能追加を繰り返す中でデータ管理が複雑化し、状態管理やデータ構造の整理に最も苦労しました。

この経験を通じて、将来的な機能追加を見据えた設計の重要性を学ぶことができました。
